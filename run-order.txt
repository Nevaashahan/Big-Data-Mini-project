1. Start Docker services
   docker compose up -d

2. Run Spring Boot
   cd springboot-app
   mvn spring-boot:run

3. Run the producer
   cd producer
   pip install -r requirements.txt
   python clickstream_producer.py

4. Run the Spark job locally with Python dependencies installed
   cd spark-jobs
   pip install -r requirements.txt
   spark-submit \
     --packages org.apache.spark:spark-sql-kafka-0-10_2.12:3.5.1 \
     stream_processor.py

5. Open Airflow and trigger the DAG
   URL: http://localhost:8080
   Login: admin / admin

6. Check Spring Boot endpoints
   http://localhost:8088/health
   http://localhost:8088/products
   http://localhost:8088/alerts
   http://localhost:8088/reports/top-products?reportDate=2026-04-15
